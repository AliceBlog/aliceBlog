##Aliceの个人小屋
用于分享、记录生活中的点点滴滴

前端采用angularjs2，后台管理采用react

##如何让项目奔跑？

* npm install -g typings
* typings install
* npm install
* npm run start
> **注意：**node-sass在第一次install的时候不一定能成功，如果运行时报node-sass的错误，那...你毫无选择，再装一次**。
